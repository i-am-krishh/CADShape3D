# CADShape3d 🚀

A simple CAD-like application built using Qt + OpenGL.

## Features
- 2D Shape Editor (Square, Rectangle, Circle)
- 3D Visualization (Cube, Cuboid, Sphere)
- Mouse-based interaction (drag, scale)
- Shape selection and deletion

## Controls
- Press 1 → Square
- Press 2 → Rectangle
- Press 3 → Circle
- Double click → Add shape
- Drag → Move / Scale
- Delete → Remove shape

## Tech Stack
- C++
- Qt (QOpenGLWidget)
- OpenGL
